# Rat in Fridge Simulator

## Overview
A browser-based Phaser 3 game where you play as a rat exploring a fridge. This is a static HTML/CSS/JavaScript game.

## Project Structure
- `index.html` - Main entry point
- `css/style.css` - Styling
- `js/main.js` - Phaser game configuration
- `js/scenes/` - Game scenes (TitleScene, StoryScene, GameScene)
- `assets/images/` - Game sprites and images

## Technology Stack
- Phaser 3.80.1 (loaded via CDN)
- Vanilla JavaScript
- Static HTML/CSS

## Running Locally
The project is served via Python's HTTP server on port 5000.

## Deployment
Configured as a static site deployment serving from the root directory.
